# Deloitte Data Analytics Job Simulation (Forage)

## 📊 Project Overview
This project is part of the Deloitte Data Analytics Job Simulation on Forage. It involves solving real-world business problems using data analysis and visualization techniques.

## 🛠️ Tasks Performed
- Cleaned and processed raw datasets
- Analyzed business data using Excel
- Identified key trends and patterns
- Created interactive dashboards using Tableau
- Generated insights for business decision-making

## 📈 Skills Gained
- Data Cleaning & Preprocessing
- Data Analysis
- Data Visualization (Tableau)
- Problem Solving
- Business Insights

## 🧰 Tools Used
- Microsoft Excel
- Tableau

## 📌 Key Insights
- Identified trends in customer data
- Highlighted business improvement areas
- Supported data-driven decisions
